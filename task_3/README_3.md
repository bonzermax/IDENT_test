**Задание 3.1**
- Аннотация о возвращаемом типе данных мало говорит о том, что действительно вернётся, правильнее использовать list[PatientORM]  
- Перед .all() лучше использовать .scalars(), чтобы получить результат сразу в формате list[PatientORM]  
- Обработка ошибок слишком общая  
- Если репозиториев будет много или планируется, что в проекте появится другой тип БД, можно реализовать сначала абстрактный репозиторий(ABCRepository), от которого будет наследоваться SQLAlchemyRepository – для взаимодействия через SQLAlchemy и уже от него все остальные репозитории, которые взаимодействуют с ORM
